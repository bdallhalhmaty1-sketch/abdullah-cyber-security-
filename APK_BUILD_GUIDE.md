# كيف تحصل على ملف APK لتطبيق "عبدالله الحماطي - الأمن السيبراني"

هذا الدليل يوضّح كيف تبني ملف APK قابل للتثبيت المباشر على أجهزة أندرويد، من خارج بيئة Replit، باستخدام خدمة بناء سحابية مجانية من Expo (لا تحتاج لتثبيت Android Studio).

## 1. تحميل المشروع من Replit

1. من داخل Replit، افتح قائمة الملفات (Files).
2. اضغط على النقاط الثلاث (⋮) بجانب اسم المشروع في الأعلى.
3. اختر **Download as zip** لتحميل نسخة كاملة من المشروع إلى جهازك.
4. فك ضغط الملف (extract) في أي مجلد على جهازك.

> ملاحظة: هذا المشروع عبارة عن "مونوريبو" (monorepo) — أي أن تطبيق الأمن السيبراني هو أحد عدة مشاريع داخل نفس المستودع، موجود في مسار `artifacts/cyber-shield`. يجب تحميل المشروع كاملًا (وليس هذا المجلد فقط) لأن بعض الإعدادات المشتركة موجودة في الجذر.

## 2. تجهيز جهازك (مرة واحدة فقط)

تحتاج إلى تثبيت:

- **Node.js** (الإصدار 20 أو أحدث) من [nodejs.org](https://nodejs.org)
- **pnpm** — بعد تثبيت Node.js، افتح الطرفية (Terminal / Command Prompt) ونفّذ:
  ```
  npm install -g pnpm
  ```
- **حساب Expo مجاني** — أنشئه من [expo.dev](https://expo.dev/signup) إذا لم يكن لديك حساب.
- **أداة EAS CLI** (أداة البناء السحابي من Expo):
  ```
  npm install -g eas-cli
  ```

## 3. تثبيت اعتماديات المشروع

من الطرفية، انتقل إلى مجلد المشروع الذي فككت ضغطه، ثم نفّذ من **جذر المشروع** (المجلد الرئيسي، لا مجلد `artifacts/cyber-shield`):

```
pnpm install
```

انتظر حتى ينتهي التثبيت (قد يأخذ بضع دقائق).

## 4. تسجيل الدخول إلى Expo

```
eas login
```

أدخل بيانات حساب Expo الذي أنشأته في الخطوة 2.

## 5. الانتقال إلى مجلد التطبيق

```
cd artifacts/cyber-shield
```

## 6. تجهيز إعدادات البناء

```
eas build:configure
```

اختر **Android** عند السؤال عن المنصة. سينشئ هذا ملف `eas.json` داخل المجلد.

## 7. تعديل نوع الملف الناتج إلى APK مباشر

بشكل افتراضي، Expo يبني ملف `.aab` (مخصص لمتجر Google Play وليس للتثبيت المباشر). لإنتاج ملف `.apk` قابل للتثبيت مباشرة على أي جهاز أندرويد، افتح ملف `eas.json` الذي تم إنشاؤه وأضف السطر التالي داخل قسم `"preview"` (أو أنشئه إذا لم يكن موجودًا):

```json
{
  "build": {
    "preview": {
      "android": {
        "buildType": "apk"
      }
    }
  }
}
```

## 8. بدء البناء

```
eas build --platform android --profile preview
```

- سيرفع الأمر المشروع إلى خوادم Expo السحابية ويبنيه هناك (لا حاجة لأي برنامج إضافي على جهازك).
- البناء يستغرق عادة بين 10 و20 دقيقة.
- بعد الانتهاء، ستظهر لك رابط مباشر لتحميل ملف `.apk` في الطرفية، وأيضًا يمكنك إيجاده في لوحة تحكم حسابك على [expo.dev](https://expo.dev) تحت قسم Builds.

## 9. تثبيت التطبيق على جهاز أندرويد

1. حمّل ملف `.apk` على هاتفك (عبر الرابط مباشرة أو بنقله من الحاسوب).
2. عند فتح الملف، سيطلب منك أندرويد تفعيل خيار **"السماح بالتثبيت من مصادر غير معروفة"** (Install from unknown sources) — وافق عليه لهذا التطبيق فقط.
3. أكمل التثبيت — سيظهر التطبيق في قائمة التطبيقات على هاتفك بشكل طبيعي.

---

**ملخص سريع للأوامر بالترتيب:**

```
npm install -g pnpm eas-cli
# (بعد تفكيك ضغط المشروع والانتقال إلى جذره)
pnpm install
eas login
cd artifacts/cyber-shield
eas build:configure
# عدّل eas.json كما في الخطوة 7
eas build --platform android --profile preview
```
